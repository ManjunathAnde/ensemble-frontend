# ❤️  Ensemble Frontend

Frontend application for **Ensemble**, a cardiovascular disease risk prediction platform powered by XGBoost and SHAP explainability.

Built with React, Vite, Tailwind CSS, and Framer Motion, the frontend provides an interactive workflow for collecting patient data, submitting predictions, and visualizing model outputs through risk gauges and explainability dashboards.

## 🛠 Tech Stack

* React
* Vite
* Tailwind CSS
* Framer Motion
* Axios

## Features

* Modern landing page
* Clinical assessment form
* Client-side validation
* Real-time API integration
* Animated risk visualization
* SHAP explainability dashboard
* Responsive design

## Local Setup

```bash
git clone https://github.com/ManjunathAnde/ensemble-frontend
cd ensemble-frontend

npm install
```

Create a `.env` file:

```env
VITE_API_URL=http://localhost:8000
```

Run:

```bash
npm run dev
```
